# WordPress AI SEO 插件

利用 **DeepSeek** / **NVIDIA GLM** 大模型，一键为 WordPress 文章生成 SEO 标题、关键词、描述、标签及深度正文内容。

---

## 功能概览

| 功能 | 触发方式 | 说明 |
|------|---------|------|
| AI 生成 SEO 标题 | 手动点击按钮 | 生成 15–30 字符的 SEO 优化标题 |
| AI 生成关键词 | 手动点击按钮 | 生成逗号分隔的 SEO 关键词组 |
| AI 生成描述 | 手动点击按钮 | 生成 50–150 字符的 Meta Description |
| AI 推荐标签 | 手动点击按钮 | 生成 8–12 个可勾选的文章标签 |
| AI 生成正文 | 手动点击按钮 | 生成含表格、FAQ 的深度中文正文，替换编辑器内容 |
| 参考 URL 抓取 | 自动（生成时） | 抓取指定页面内容作为 AI 创作上下文 |

---

## 系统要求

| 项目 | 要求 |
|------|------|
| WordPress | 5.8 及以上 |
| PHP | 7.4 及以上（推荐 8.1+） |
| API 账号 | DeepSeek 或 NVIDIA NGC 账号（至少一个） |

## 安装方法

### 方法一：手动上传（推荐）

1. 下载本插件的 ZIP 包，或克隆仓库：
   ```bash
   git clone https://github.com/itgoyo/wordpress-ai-seo.git
   ```
2. 将整个 `wordpress-ai-seo` 文件夹上传到 WordPress 目录：
   ```
   wp-content/plugins/wordpress-ai-seo/
   ```
3. 登录 WordPress 后台 → **插件** → 找到「WordPress AI SEO」→ 点击**启用**

### 方法二：ZIP 上传

1. 将项目打包为 `wordpress-ai-seo.zip`
2. 后台 → **插件** → **安装插件** → **上传插件** → 选择 ZIP → **立即安装** → **启用**

## 配置 API Key

启用插件后，进入 **WordPress 后台 → 设置 → AI SEO** 进行配置。

### DeepSeek 配置

| 配置项 | 说明 | 示例 |
|-------|------|------|
| DeepSeek API Key | 在 [platform.deepseek.com](https://platform.deepseek.com) 申请 | `sk-xxxxxxxxxxxxxxxx` |
| DeepSeek 模型 | 默认 `deepseek-chat`，也可填 `deepseek-reasoner` | `deepseek-chat` |

> **获取方式：** 注册 DeepSeek 账号 → 控制台 → API Keys → 创建密钥

### NVIDIA GLM 配置

| 配置项 | 说明 | 示例 |
|-------|------|------|
| NVIDIA GLM API Key | 在 [build.nvidia.com](https://build.nvidia.com) 免费申请 | `nvapi-xxxxxxxx` |
| GLM 模型 | 默认 `nvidia/glm-4-9b-chat` | `nvidia/glm-4-9b-chat` |

> **获取方式：** 注册 NVIDIA NGC 账号 → 进入 NIM 平台 → 生成 API Key（有免费额度）

### 其他配置

| 配置项 | 说明 | 可选值 |
|-------|------|------|
| 使用模型 | 选择当前使用哪个模型 | DeepSeek / NVIDIA GLM |
| 正文目标字数 | 生成正文的目标长度（默认 1500 字） | 800 / 1500 / 2500 / 4000 |

## 使用教程

### 第一步：进入文章编辑页

新建文章或编辑已有文章，填写**文章标题**（这是 AI 生成的主要依据）。

---

### 第二步：填写参考站点 URL（可选）

在右侧「🤖 AI SEO」面板中，填入**参考站点 URL**。

```
例如：https://example.com/some-article
```

AI 将自动抓取该页面的文字内容（最多 3000 字）作为创作上下文，使生成内容与参考站点相关。**留空则仅凭标题生成。**

---

### 第三步：AI 生成 SEO 信息

点击「✨ AI 生成 SEO 信息」按钮：

1. AI 生成**标题 / 关键词 / 描述**的预览卡片
2. 确认内容后点「**✅ 应用到字段**」，自动填入对应输入框
3. 不满意可点「**✖ 放弃**」重新生成

> **字段说明：**
> - 标题：建议 15–30 字符，会影响搜索结果标题栏显示
> - 关键词：多个关键词用英文逗号 `,` 分隔
> - 描述：建议 50–150 字符，会影响搜索结果摘要显示

---

### 第四步：AI 推荐标签

点击「🏷️ AI 推荐标签」按钮：

1. AI 生成 8–12 个标签，以**勾选框**形式展示
2. 勾选需要的标签，点「**✅ 添加选中标签**」
3. 标签会自动添加到右侧「网址标签」区域

---

### 第五步：AI 生成正文

在主编辑区上方的「🤖 AI 正文生成」面板中：

1. 选择**目标字数**（默认 1500 字）
2. 点击「**📝 AI 生成正文（替换）**」
3. 确认弹窗后，AI 开始生成（进度条显示进度）
4. 生成完成后，正文自动填入编辑器（**替换**原有内容）

#### 生成正文的结构

AI 生成的正文固定包含以下 7 个部分：

| 序号 | 结构 | 说明 |
|------|-----|------|
| 1 | 引言 | 介绍主题背景与读者痛点（约 150 字） |
| 2 | 核心概念 | 含数据对比表格 |
| 3 | 功能/特点分析 | 至少 3 个要点，每点 200 字以上 |
| 4 | 横向对比表 | 至少 3 个维度的功能对比表格 |
| 5 | 使用场景 | 含实际案例分析 |
| 6 | FAQ | 常见问题解答，至少 4 个问答 |
| 7 | 总结 | 总结与建议（约 100 字） |

> 生成内容包含至少 **2 个 Markdown 表格**，由插件自动转换为 HTML 注入编辑器。

## 常见问题（FAQ）

**Q：点击生成按钮没有反应？**
> 请检查：① 浏览器控制台是否有 JS 报错；② API Key 是否已在设置页填写并保存；③ 文章标题是否已填写。

**Q：提示「API Key 未配置」？**
> 前往 **设置 → AI SEO**，填入对应模型的 API Key 并点击「保存设置」。

**Q：生成超时或失败？**
> DeepSeek/GLM 接口调用超时设置为 120 秒。若网络环境访问 API 较慢，可尝试：① 切换另一个模型；② 减少目标字数；③ 检查服务器出口网络是否能访问 API 端点。

**Q：生成的正文如何与 Yoast SEO / RankMath 配合使用？**
> 本插件生成的 SEO 标题/关键词/描述存储在自定义 meta 字段（`_wp_ai_seo_title` 等），需在主题 `<head>` 中手动调用，或配置 Yoast SEO 使用相同字段名。建议直接将生成内容复制到 Yoast/RankMath 的对应输入框中。

**Q：正文生成后，能否撤销恢复原有内容？**
> 块编辑器（Gutenberg）支持 `Ctrl+Z` 撤销。经典编辑器不支持自动撤销，建议生成前先备份原有内容。

**Q：支持批量生成吗？**
> 当前版本仅支持单篇文章生成，不支持批量操作。

## 安全说明

- API Key 存储于 WordPress `wp_options` 表，已做 `sanitize_text_field` 处理
- 所有 AJAX 请求均验证 **WordPress Nonce** 和用户权限（`edit_posts`）
- URL 抓取功能内置 **SSRF 防护**，拒绝访问 `localhost`、`127.0.0.1`、私有 IP 等内网地址
- 所有用户输入均经过 WordPress 标准 `sanitize_*` 函数处理
